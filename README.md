# Noll Raycast Extension

Translate screenshots instantly with [Noll](https://noll.to).

## Features

- Translate screenshots directly from your clipboard
- Supports multiple target languages
- Preserves image layout and formatting
- Copy translated image to clipboard with Enter

## Usage

1. Take a screenshot with `Cmd+Shift+4`
2. Open Raycast and search for "Translate Screenshot"
3. Sign in to your Noll account (first time only)
4. Wait for translation to complete
5. Press Enter to copy the translated image

## Configuration

- **Default Target Language**: Set your preferred translation target language in extension preferences

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev
```

## Requirements

- A [Noll](https://noll.to) account
- macOS with Raycast installed
